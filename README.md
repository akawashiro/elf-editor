# elfeditor

`elfeditor` is a JSON based binary editor for ELF files. You can edit your ELF files using your favorite editor via a JSON file.

## How to install
```
git clone https://github.com/akawashiro/elfeditor.git
cd elfeditor
mkdir build
cmake ..
make install
```
